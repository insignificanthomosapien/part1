Contains three React projects:

1. Anecdotes: An anecdote generator with randomized generation and a voting option which shows the highest voted anecdotes amongst all anecdotes.
2. Course Info: A single page application containing course information about the FullStack Open course at the University of Helsinki.
3. UnicafeL A feedback application take input from users about what they thought of the coffee and services at the cafe. Ratings are shown with statistical operations such as mean, positive, negative, neutral, percentage of positive, etc.
